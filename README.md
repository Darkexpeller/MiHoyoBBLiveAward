# 原神绝区零B站直播激励抢码工具

BBLiveAward.py从服务器获取task_id以及拥有版本更新功能

BBLiveAward_offline.py 为离线版本，需要手动输入task_id


## 参数
task_id为激励奖励页面的task_id参数，每个奖励唯一

cookie需要F12从网络请求中复制完整的cookie到本工具中(F12控制台输出的doucument.cookie不完整)：
![](attachments/Pasted%20image%2020260603054641.png)

并发请求不建议延迟太低，容易被风控。如若被风控需要重新登录，并输入新的cookie
